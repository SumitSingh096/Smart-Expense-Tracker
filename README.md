# 💰 Smart Expense Tracker

A modern web-based application to track income and expenses with filtering, analytics, and visual insights.

---

## 📌 Features

* ➕ Add income and expense transactions
* ✏️ Edit and delete transactions
* 📊 Visual charts (Bar & Doughnut) for insights
* 🔍 Filter transactions by:

  * Category
  * Date
  * Type (Income/Expense)
  * Amount
* 💾 Data stored using Local Storage (no backend required)
* 📱 Responsive design (mobile-friendly)

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox, Responsive Design)
* JavaScript (Vanilla JS)
* Chart.js (for graphs)

---

## ⚙️ How It Works

1. Add a transaction (amount, category, date, type)
2. Data is saved in browser local storage
3. Transactions are displayed in a table
4. Charts update dynamically based on data
5. Use filters to search specific transactions
